# 个人主页系统

一个基于原生 HTML/CSS/JS 的极简静态主页。

Shuxuan负责叽里咕噜描绘想象，Gemini&Claude&ChatGPT负责干活()

## 技术栈

- HTML5 — 语义化结构
- CSS3 — CSS 变量、Scroll Snap、玻璃拟态
- JavaScript — 原生 ES6+，零依赖
- marked.js — Markdown 解析

## 特色

1. 自动暗黑模式，支持手动切换
2. 完全响应式布局
3. 自动更新文章、作品列表
4. 零框架，极致性能，适配GitHub Pages
5. 极简设计，大间距排版
